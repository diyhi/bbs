页面访问 - AJAX方式
请求路径 user/control/receiveRedEnvelopeList.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 页码 变量：page
类型：Integer
是否必填：否
请求成功返回 JSON格式值 类型:PageView 值内容说明：收红包分页
分页 PageView
变量 类型 说明
records List<ReceiveRedEnvelope> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
收红包 ReceiveRedEnvelope
变量 类型 说明
id String Id
giveUserId Long 发红包的用户Id
giveUserName String 发红包的用户名称
giveAccount String 发红包的账号
giveAvatarPath String 发红包的用户呢称
giveAvatarPath String 发红包的用户头像路径 页面上显示200*200头像示例${receiveRedEnvelope.giveAvatarPath}${receiveRedEnvelope.giveAvatarName}   显示100*100头像示例${receiveRedEnvelope.giveAvatarPath}100x100/${receiveRedEnvelope.giveAvatarName}
giveAvatarName String 发红包的用户头像名称
receiveRedEnvelopeId String 收红包Id
receiveUserId Long 收红包的用户Id
receiveTime Date 收取时间
amount BigDecimal 红包金额
