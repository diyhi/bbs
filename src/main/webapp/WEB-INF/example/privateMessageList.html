页面访问
请求路径 user/control/privateMessageList.htm
请求方式 GET
请求参数 页码 变量：page
类型：Integer
是否必填：否
返回值 私信分页 变量：pageView
类型：PageView
页面访问 - AJAX方式
请求路径 user/control/privateMessageList.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 页码 变量：page
类型：Integer
是否必填：否
请求成功返回 JSON格式值 类型:PageView 值内容说明：私信分页
分页 PageView
变量 类型 说明
records List<PrivateMessage> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
私信 PrivateMessage
变量 类型 说明
id String Id
friendUserId Long 私信对方用户Id
friendUserName String 私信对方用户名称
friendAccount String 私信对方账号
friendNickname String 私信对方用户呢称
friendAvatarPath String 私信对方头像路径 页面上显示200*200头像示例${privateMessage.friendAvatarPath}${privateMessage.friendAvatarName}   显示100*100头像示例${privateMessage.friendAvatarPath}100x100/${privateMessage.friendAvatarName}
friendAvatarName String 私信对方头像名称
senderUserId Long 发送者用户Id
senderUserName String 发送者用户名称
senderAccount String 发送者账号
senderNickname String 发送者用户呢称
senderAvatarPath String 发送者头像路径 页面上显示200*200头像示例${privateMessage.senderAvatarPath}${privateMessage.senderAvatarName}   显示100*100头像示例${privateMessage.senderAvatarPath}100x100/${privateMessage.senderAvatarName}
senderAvatarName String 发送者头像名称
messageContent String 消息内容
status Integer 消息状态 10:未读 20:已读
sendTime Date 发送时间
readTime Date 阅读时间
