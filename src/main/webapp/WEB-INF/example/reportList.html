页面访问
请求路径 user/control/reportList.htm
请求方式 GET
请求参数 页码 变量：page
类型：Integer
是否必填：否
返回值 用户举报分页 变量：pageView
类型：PageView
页面访问 - AJAX方式
请求路径 user/control/reportList.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 页码 变量：page
类型：Integer
是否必填：否
请求成功返回 JSON格式值 类型:PageView 值内容说明：用户举报分页
分页 PageView
变量 类型 说明
records List<Report> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
用户举报 Report
变量 类型 说明
id String Id
userId String 用户Id
reportTypeId String 举报分类Id
reportTypeName String 举报分类名称
parameterId String 参数Id (话题Id； 评论Id； 评论回复Id； 问题Id； 答案Id； 答案回复Id)
extraParameterId String 扩展参数Id (模块值10：空； 模块值20：话题Id； 模块值30：话题Id-评论Id； 模块值40：空； 模块值50：问题Id； 模块值60：问题Id-答案Id)
module Integer 模块 10:话题 20:评论 30:评论回复 40:问题 50:答案 60:答案回复
reason String 举报理由
processResult String 处理结果
imageInfoList List<ImageInfo> 图片
postTime Date 举报时间
processCompleteTime Date 处理完成时间 处理状态(40:投诉失败 50:投诉成功)为处理完成状态
status Integer 处理状态 10:待处理 40:投诉失败 50:投诉成功 1010:待处理【用户删除】 1020:需补充理由【用户删除】 1030:已补充理由【用户删除】 1040:投诉失败【用户删除】 1050:投诉成功【用户删除】 100010:待处理【员工删除】 100020:需补充理由【员工删除】 100030:已补充理由【员工删除】 100040:投诉失败【员工删除】 100050:投诉成功【员工删除】
图片信息 ImageInfo
变量 类型 说明
path String 图片路径
name String 图片名称
