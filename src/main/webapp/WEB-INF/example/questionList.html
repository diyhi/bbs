页面访问 - AJAX方式
请求路径 user/control/questionList.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 参数表
请求返回 JSON格式值 类型:PageView
分页 PageView
变量 类型 说明
records List<Question> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
问题 Question
变量 类型 说明
id Long 问题Id
title String 标题
questionTagAssociationList List<QuestionTagAssociation> 问题标签关联集合
content String 问题内容
summary String 内容摘要
postTime Date 发表时间
lastAnswerTime Date 最后回答时间
lastUpdateTime Date 最后修改时间
adoptionAnswerId Date 采纳的答案Id
answerTotal Long 答案总数
allow boolean 允许回答
viewTotal Long 查看总数
userName String 用户名称
isStaff boolean 是否为员工
sort boolean 排序
status Integer 状态 10.待审核 20.已发布 110.待审核删除 120.已发布删除
amount BigDecimal 悬赏金额
point Long 悬赏积分
问题标签关联 QuestionTagAssociation
变量 类型 说明
questionTagId Long 问题标签Id
questionTagName String 问题标签名称
请求参数
参数 类型 是否必填 说明
page Integer 否 页码
