页面访问 - AJAX方式

请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 参数表
请求返回 JSON格式值 类型:PageView
分页 PageView
变量 类型 说明
records List<Like> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
点赞 Like
变量 类型 说明
id String 点赞Id
userName String 点赞的用户名称
account String 账号
nickname String 呢称
avatarPath String 头像路径
avatarName String 头像名称
postUserName String 发布话题的用户名称
addtime Date 加入时间
topicId Long 话题Id
topicTitle String 话题标题

参数 类型 是否必填 说明

示例

提交 删除点赞 - 默认方式
请求路径 user/control/deleteLike.htm
请求方式 POST
请求参数 参数表
提交成功返回 默认跳转到jump模板页显示提交成功信息；跳转路径参数jumpUrl有值时按参数值路径跳转
提交失败返回 跳转到message模板页显示提示信息
提交 删除点赞 - AJAX方式
请求路径 user/control/deleteLike.htm
请求方式 POST
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 参数表
提交返回 JSON格式值 类型:Map<String,Object> 键参数：success
值内容说明：成功返回true 失败返回false
值类型：String
键参数：error
值内容说明：错误值集合
值类型：Map<String,String>
删除点赞 错误值 error
变量 类型 说明
like String 点赞错误
token String 令牌错误
提交'删除点赞'参数
参数 类型 是否必填 说明
likeId String 是 点赞Id
token String 是 令牌
jumpUrl String 否 '提交表单 - 默认方式'提交成功后跳转路径,不填写默认跳转到表单提交页；参数必须经过encodeBase64URL编码后才能提交。例${encodeBase64URL('/index')}
