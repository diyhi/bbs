页面访问
请求路径 user/control/userLoginLogList.htm
请求方式 GET
请求参数 页码 变量：page
类型：Integer
是否必填：否
返回值 用户登录日志分页 变量：pageView
类型：PageView
页面访问 - AJAX方式
请求路径 user/control/userLoginLogList.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 页码 变量：page
类型：Integer
是否必填：否
请求成功返回 JSON格式值 类型:PageView 值内容说明：用户登录日志分页
分页 PageView
变量 类型 说明
records List<UserLoginLog> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
用户登录日志 UserLoginLog
变量 类型 说明
id String Id
userId String 用户Id
logonTime Date 登录时间
typeNumber Integer 类型编号 10:登录 20:续期
ipAddress String IP归属地
