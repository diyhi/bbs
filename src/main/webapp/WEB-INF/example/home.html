页面访问
请求路径 user/control/home.htm
请求方式 GET
请求参数 用户名称 变量：userName
类型：String
是否必填：按需填写
返回值 用户 变量：user
类型：User
页面访问 - AJAX方式
请求路径 user/control/home.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 用户名称 变量：userName
类型：String
是否必填：按需填写
请求成功返回 JSON格式值 类型:Map<String,Object> 键参数：user
值内容说明：用户
值类型：User
用户 user
变量 类型 说明
id long Id 值为-1时为员工未登录会员允许显示字段
userName String 会员用户名 未登录会员允许显示字段
account String 账号 未登录会员允许显示字段
state Integer 用户状态 1:正常用户 2:禁止用户 未登录会员允许显示字段
email String 邮箱地址
issue String 密码提示问题
registrationDate Date 注册日期
point Long 当前积分
gradeId Integer 等级Id 未登录会员允许显示字段
gradeName String 等级名称 未登录会员允许显示字段
userRoleNameList List<String> 用户角色名称集合未登录会员允许显示字段
mobile String 绑定手机
realNameAuthentication boolean 是否实名认证
ipAddress String IP归属地
avatarPath String 头像路径 未登录会员允许显示字段 页面上显示200*200头像示例${user.avatarPath}${user.avatarName}   显示100*100头像示例${user.avatarPath}100x100/${user.avatarName}
avatarName String 头像名称 未登录会员允许显示字段
