页面访问 - AJAX方式
请求路径 user/control/balance.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 页码 变量：page
类型：Integer
是否必填：否
请求成功返回 JSON格式值 类型:Map<String,Object> 键参数：pageView
值内容说明：支付日志分页
值类型：PageView
键参数：deposit
值内容说明：预存款
值类型： BigDecimal
分页 PageView
变量 类型 说明
records List<PaymentLog> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
支付日志 PaymentLog
变量 类型 说明
paymentRunningNumber String 支付流水号
paymentModule Integer 支付模块 1.订单支付 5.用户充值 6.账户提现 70.余额购买话题隐藏内容 80.解锁话题隐藏内容分成 90.悬赏金额 100.采纳答案 110.调整赏金 120.话题发红包 130.话题收红包 140.话题返还红包
sourceParameterId Long 来源参数Id 用户Id 话题Id 评论Id 问题Id 答案Id 发红包Id 收红包Id
interfaceProduct Integer 接口产品 -1:员工操作 0:预存款支付 1.支付宝即时到账 4.支付宝手机网站
tradeNo String 交易号
operationUserType Integer 操作用户类型 0:系统 1: 员工 2:会员
operationUserName String 操作用户名称
userName String 用户名称
amountState int 金额状态 1:账户存入 2:账户支出
amount BigDecimal 金额 精度为12位，小数点位数为2位
times Date 时间
remark String 备注
