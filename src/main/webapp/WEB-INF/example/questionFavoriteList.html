页面访问 - AJAX方式
请求路径 user/control/questionFavoriteList.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 参数表
请求返回 JSON格式值 类型:PageView
分页 PageView
变量 类型 说明
records List<Favorites> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
收藏夹 Favorites
变量 类型 说明
id String 收藏夹Id
userName String 用户名称
account String 账号
addtime Date 加入时间
questionId Long 问题Id
questionTitle String 问题标题
请求参数
参数 类型 是否必填 说明
questionId Long 是 问题Id
page Integer 否 页码
