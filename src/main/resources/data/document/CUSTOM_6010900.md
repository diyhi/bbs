-  **接口说明：** 修改话题表单
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果) 



### 请求参数表
| 字段         | 类型         | 是否必填          | 说明 |
|:-----------|:-----------|:------------|:---|
| `topicId`      | Long       |  是  | 话题Id  |


### 返回结果
| 字段 | 类型                                                 | 描述          | 备注                                                           |
| :--- |:---------------------------------------------------|:------------|:-------------------------------------------------------------|
| `captchaKey` | String                                             | 验证码编号 | 根据'全站设置 - 基本设置 - 发表话题每分钟提交超过'参数计算是否含有值                       |
| `allowTopic` | boolean                                            |是否允许提交话题 |                                                              |
| `userGradeList` | JSON格式[List&lt;UserGrade>](#anchor:用户等级-UserGrade) |用户等级集合 |                                                              |
| `topic` | [Topic](#anchor:话题-Topic)                          |话题 |                                                              |
| `voteTheme` | [VoteTheme](#anchor:投票主题-VoteTheme)                |投票主题 |                                                              |
| `maxVoteOptions` | Integer                                            |发起投票选项数量 |        空值为不限制； 0为不允许发起投票                                                      |
| `fileSystem` | int                                                | 文件系统 | 0.本地 10.SeaweedFS 20.MinIO 30.阿里云OSS                         |
| `supportEditor` | int                                                | 支持编辑器 | 10.仅富文本编辑器 20.仅Markdown编辑器  30.富文本编辑器优先 40.Markdown编辑器优先     |
| `availableTag` | JSON格式List&lt;String>                              | 话题编辑器允许使用标签(配合KindEditor编辑器使用) |     |
