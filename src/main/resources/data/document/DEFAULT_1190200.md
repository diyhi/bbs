-  **接口说明：** 私信对话列表
-  **接口地址：** /user/control/privateMessageChatList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型      | 是否必填 | 说明 |
|:-----------|:--------|:-----|:---|
| `page`      | Integer | 否 | 页码  |
| `friendUserName`      | String  | 是    | 对方用户名称  |



### 返回结果
| 字段 | 类型                                             | 描述     | 备注                                                      |
| :--- |:-----------------------------------------------|:-------|:--------------------------------------------------------|
| `pageView` | [PageView&lt;PrivateMessage>](#anchor:分页-PageView) | 私信对话分页 |                                                         |
| `chatUser` | [User>](#anchor:用户-User)                   | 对话用户   |  |




### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;PrivateMessage>](#anchor:私信-PrivateMessage)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |
