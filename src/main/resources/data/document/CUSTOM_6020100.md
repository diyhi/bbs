-  **接口说明：** 问题分页
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 自定义 或 [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Question>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |备注                                   |
|:-----------|:-------|:-----|:---|:---|
| `page`      | Integer | 否 | 页码  ||
| `questionTagId`      | Long | 否 | 标签Id  ||
| `filterCondition`      | Long | 否 | 过滤条件  |全部：值为空或10   未解决：20   已解决：30   积分悬赏：40   现金悬赏：50|

### 分页 `PageView`
| 字段 | 类型                                       | 描述          | 备注                                   |
| :--- |:-----------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Question>](#anchor:问题-Question) | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)      | 分页索引 |                                      |
| `totalpage` | long                                     | 总页数 |                                      |
| `maxresult` | long                                     | 每页显示记录数 |                                      |
| `currentpage` | long                                     | 当前页 |                                      |
| `pagecount` | long                                     | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 问题 `Question`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | Long          | 问题Id    |                    |
| `title` | String        | 标题      |                        |
| `questionTagAssociationList` | [List&lt;QuestionTagAssociation>](#anchor:问题标签关联-QuestionTagAssociation)           | 问题标签关联集合    |                   |
| `summary` | String        | 内容摘要    |                       |
| `postTime` | LocalDateTime | 发表时间    |                        |
| `lastAnswerTime` | LocalDateTime | 最后回答时间    |                        |
| `answerTotal` | Long          | 答案总数      |                        |
| `allow` | boolean       | 允许回答      |                        |
| `viewTotal` | Long          | 查看总数      |                        |
| `userName` | String        | 用户名称      |                        |
| `account` | String        | 账号      |                        |
| `userRoleNameList` | List&ltString>        | 用户角色名称集合     |                                                                                                                                                                                                                                                                                                                                                          |
| `nickname` | String        | 呢称      |                        |
| `userInfoStatus` | Integer        | 用户信息状态      |       -30.账号已注销(不显示数据) -20.账号已逻辑删除(不显示数据) -10.账号已禁用(不显示数据)  0.正常 10.账号已禁用(显示数据) 20.账号已逻辑删除(显示数据)                 |
| `avatarPath` | String        | 头像路径      |     页面上显示200 &times; 200头像示例{{question.avatarPath}}{{question.avatarName}}  <br>显示100 &times; 100头像示例{{question.avatarPath}}100x100/{{question.avatarName}}                    |
| `avatarName` | String        | 头像名称      |                        |
| `isStaff` | Boolean        | 是否为员工      |                        |
| `sort` | Integer       | 排序      |            默认为0            |
