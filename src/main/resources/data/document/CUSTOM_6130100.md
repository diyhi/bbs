-  **接口说明：** 添加举报表单
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 无
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果) 



### 返回结果
| 字段 | 类型                                             | 描述         | 备注                                                           |
| :--- |:-----------------------------------------------|:-----------|:-------------------------------------------------------------|
| `captchaKey` | String                                         | 验证码编号      |         根据'全站设置 - 基本设置 - 举报每分钟提交超过'参数计算是否含有值               |
| `allowReport` | boolean                                        | 是否允许提交举报   |                                                              |
| `reportMaxImageUpload` | int                                            | 图片允许最大上传数量 |                                                              |
| `fileSystem` | int                                            | 文件系统       | 0.本地 10.SeaweedFS 20.MinIO 30.阿里云OSS                         |
| `reportTypeList` | [List&lt;ReportType>](#anchor:举报分类-ReportType) | 举报分类集合     |                                                              |
