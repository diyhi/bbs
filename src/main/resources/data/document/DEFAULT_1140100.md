-  **接口说明：** 参与投票
-  **接口地址：** /user/control/vote/addVoteRecord
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填          | 说明    |
|:-----------|:-----------|:------------|:------|
| `voteOptionId`      | String       |  是  | 投票选项Id  |


### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
