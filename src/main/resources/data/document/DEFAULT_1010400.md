-  **接口说明：** 话题解锁
-  **接口地址：** /user/control/topic/unhide
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填          | 说明 |
|:-----------|:-----------|:------------|:---|
| `topicId`      | Long       |  是  | 话题Id  |
| `hideType` | Integer    | 是 |  隐藏类型 10:输入密码可见 20:评论话题可见 30:达到等级可见 40:积分购买可见  50:余额购买可见  |
| `password` | String     | 按需填写 |  密码，当hideType值为10时必须填写  |



### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |


### 错误结果 `error`
| 字段 | 类型              | 描述          |
| :--- |:----------------|:------------|
| `topicUnhide` | String          | 话题解锁错误      | 
| `password` | String          | 密码错误        |
