-  **接口说明：** 系统通知列表
-  **接口地址：** /user/control/systemNotifyList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;SubscriptionSystemNotify>](#anchor:分页-PageView)



### 请求参数表
| 字段         | 类型      | 是否必填 | 说明 |
|:-----------|:--------|:-----|:---|
| `page`      | Integer | 否 | 页码  |





### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;SubscriptionSystemNotify>](#anchor:订阅系统通知-SubscriptionSystemNotify)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |




### 订阅系统通知 `SubscriptionSystemNotify`
| 字段 | 类型            | 描述      | 备注                                                                                                               |
| :--- |:--------------|:--------|:-----------------------------------------------------------------------------------------------------------------|
| `id` | String          | Id    |                                                                                                                                                                                                          |
| `systemNotifyId` | Long          | 系统通知Id    |                                                                                                                                                                                                          |
| `userId` | Long        | 用户Id    |                                                                                                                                                                                                          |
| `content` | String        | 通知内容    |                                                                                                                                                                                                          |
| `status` | Integer | 消息状态    |     10:未读  20:已读                                                                                                                                                                                                     |
| `sendTime` |     LocalDateTime    |   发送时间    |                                                                                                                                                                                                          |
| `readTime` |     LocalDateTime    |   阅读时间    |                                                                                                                                                                                                          |
