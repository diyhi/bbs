-  **接口说明：** 收藏夹列表
-  **接口地址：** /user/control/favoriteList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Favorites>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |



### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Favorites>](#anchor:收藏夹-Favorites)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 收藏夹 `Favorites`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | String          | 收藏夹Id    |                    |
| `module` | Integer        | 模块      |           10:话题 20:问题              |
| `userName` | String        | 用户名称      |                        |
| `account` | String          | 账号    |                        |
| `nickname` | String          |   呢称  |                        |
| `avatarPath` | String          |   头像路径  |                        |
| `avatarName` | String          |  头像名称   |                        |
| `addtime` | LocalDateTime | 加入时间    |                        |
| `topicId` | Long | 话题Id  |                        |
| `topicTitle` | String        | 话题标题      |                        |
| `questionId` | Long | 问题Id  |                        |
| `questionTitle` | String        | 问题标题      |                        |
