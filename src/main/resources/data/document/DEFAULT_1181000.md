-  **接口说明：** 找回密码第二步
-  **接口地址：** /forgotPassword/step2
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 |备注                                    |
|:-----------|:-----------|:-----|:---|:---|
| `userName`      | String       | 是    | 会员用户名  ||
| `password`      | String       | 是    | 密码  |必须执行sha256一次|
| `type`    | Integer     | 是    |  用户类型  | 10:本地账号密码用户 20: 手机用户 30: 邮箱用户 40:微信用户 80:其他用户|
| `answer`    | String     | 按需填写 | 密码提示答案   |      必须执行sha256一次                                       |
| `smsCode`    | String     | 按需填写 | 短信验证码   |                                             |
| `emailCode`    | String     | 按需填写 |  邮箱验证码  |                                             |
| `captchaKey` | String     | 按需填写 | 验证码编号 ||
| `captchaValue`  | String     | 按需填写 |  验证码值 同一编号验证码每分钟提交次数不能超过60次,超出时验证码图片返回空字符  ||






### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |
| `captchaKey` | String                                      |新验证码编号 |当`success`参数返回`false`含有值|



### 错误结果 `error`
| 字段 | 类型              | 描述     |
| :--- |:----------------|:-------|
| `captchaValue` | String          | 验证码错误  | 
| `user` | String          | 用户错误  | 
| `account` | String          | 账号错误   | 
| `password` | String          | 密码错误   | 
| `answer` | String          | 密码提示答案错误   | 
| `mobile` | String          | 手机号错误 |
| `email` | String          | 邮箱错误 |
| `smsCode` | String          | 手机号验证码错误 |
| `emailCode` | String          | 邮箱验证码错误 |
| `type` | String          | 用户类型错误 |
