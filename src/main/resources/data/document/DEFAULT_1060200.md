-  **接口说明：** 会员卡订单列表
-  **接口地址：** /user/control/membershipCardOrderList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;MembershipCardOrder>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;MembershipCardOrder>](#anchor:会员卡订单-MembershipCardOrder)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 会员卡订单 `MembershipCardOrder`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `orderId` | Long          | 订单号    |                    |
| `userName` | String        | 用户名称      |                        |
| `createDate` | LocalDateTime | 订单创建时间    |                        |
| `accountPayable` | BigDecimal        | 应付款(实际需要支付的费用)      |                        |
| `accountPoint` | Long          | 应付积分    |                        |
| `paymentAmount` | BigDecimal          |   已支付金额  |                        |
| `paymentPoint` | Long          |   已支付积分  |                        |
| `userRoleId` | String          |  购买的会员卡用户角色Id   |                        |
| `roleName` | String          |  购买的会员卡用户角色名称   |                        |
| `membershipCardId` | Long          |  购买的会员卡Id   |                        |
| `specificationId` |      Long     |  购买的会员卡规格Id   |                        |
| `specificationName` |      String     |  购买的会员卡规格名称   |                        |
| `quantity` |    Integer       |  购买的会员卡数量   |                        |
| `duration` |      Integer     |  购买的会员卡有效期时长   |                        |
| `unit` |     Integer      |  购买的会员卡时长单位    |              10.小时 20.日 30.月 40.年          |
