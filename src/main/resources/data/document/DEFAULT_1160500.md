-  **接口说明：** 支付完成通知
-  **接口地址：** /paymentCompleted/{interfaceProduct}/{paymentModule}/{parameterId}
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)


### 请求参数表
| 字段         | 类型         | 是否必填 | 说明                                                 |备注                                        |
|:-----------|:-----------|:-----|:---------------------------------------------------|:---------------------------------------------------|
| `interfaceProduct`      | Integer      | 是    | 接口产品    | 0:账户支付 1.支付宝即时到账 4.支付宝手机网站(alipay.trade.wap.pay接口) | 
| `out_trade_no`      | String       |   是   | 商户订单号    |  | 
| `trade_no`      | String       |    是  | 支付宝交易号    |  | 
| `total_amount`      | BigDecimal      |   是   | 交易金额    |  |


### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                        |
| :--- |:--------------------------------------------|:------------|:------------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |                                           |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |                                           |
