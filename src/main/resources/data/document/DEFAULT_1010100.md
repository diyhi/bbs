-  **接口说明：** 添加话题
-  **接口地址：** /user/control/topic/add
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填          | 说明 |
|:-----------|:-----------|:------------|:---|
| `tagId`      | Long       |  是  | 标签Id  |
| `title`      | String     | 是 |  标题  |
| `content`    | String     |按需填写 |  话题内容  |
| `isMarkdown` | Boolean    | 按需填写 |  是否使用Markdown  |
| `markdownContent` | String     | 按需填写 |  Markdown内容  |
| `type` | Integer    | 按需填写 | 红包类型 20.随机金额 30.固定金额 提交本参数时会检测红包参数  |
| `totalAmount` | BigDecimal | 按需填写 |  红包总金额 type=20时必填  |
| `singleAmount` | BigDecimal | 按需填写 | 单个红包金额 type=30时必填 |
| `giveQuantity` | Integer    | 按需填写 | 红包发放数量 |
| `isVote` | boolean    | 是 |  是否发起投票  |
| `voteTitle`  | String     | 按需填写 |  投票标题  |
| `voteOptionTextList`  | String[]   | 按需填写 |  投票选项文本集合  |
| `voteMaxChoice` | Integer    | 按需填写 |  投票最大可选数  |
| `voteEndDate` | String     | 按需填写 |  投票截止时间  |
| `captchaKey` | String     | 按需填写 | 验证码编号 |
| `captchaValue`  | String     | 按需填写 |  验证码值 同一编号验证码每分钟提交次数不能超过60次,超出时验证码图片返回空字符  |




### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |
| `captchaKey` | String                                      |新验证码编号 |当`success`参数返回`false`时根据'全站设置 - 基本设置 - 话题每分钟提交超过'参数计算是否含有值|


### 错误结果 `error`
| 字段 | 类型              | 描述     |
| :--- |:----------------|:-------|
| `captchaValue` | String          | 验证码错误  | 
| `topic` | String          | 话题错误   | 
| `tagId` | String          | 标签错误 |
| `title` | String          | 标题错误 |
| `content` | String          | 话题内容错误 |
| `totalAmount` | String          | 红包总金额错误 |
| `redEnvelopeLimit` | String          | 红包限制错误 |
| `giveQuantity` | String          | 红包数量错误 |
| `singleAmount` | String          | 单个红包金额错误 |
| `voteOption` | String          | 投票选项错误 |
| `vote` | String          | 发起投票错误 |
| `voteTitle` | String          | 投票标题错误 |
| `voteMaxChoice` | String          | 投票最大可选数参数错误 |
| `voteEndDate` | String          | 截止日期错误 |
