-  **接口说明：** 话题明细
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 自定义 或 [参数表](#anchor:请求参数表)
-  **返回类型：** [Topic](#anchor:话题-Topic)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `topicId`      | Long | 是    | 话题Id  |

| 字段 | 类型            | 描述      | 备注                                                                                                                                                                                                                                                                                                                                                       |
| :--- |:--------------|:--------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `id` | Long          | 话题Id    |                                                                                                                                                                                                                                                                                                                                                          |
| `title` | String        | 标题      |                                                                                                                                                                                                                                                                                                                                                          |

| `lastUpdateTime` | LocalDateTime | 最后修改时间    |                                                                                                                                                                                                                                                                                                                                                          |
| `commentTotal` | Long          | 评论总数      |                                                                                                                                                                                                                                                                                                                                                          |
