-  **接口说明：** 会员注册表单
-  **接口地址：** /register
-  **请求方式：** GET
-  **请求参数：** 无
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)





### 返回结果
| 字段 | 类型                                                   | 描述      | 备注                                   |
| :--- |:-----------------------------------------------------|:--------|:-------------------------------------|
| `captchaKey` | String                                               | 验证码编号    | 根据'全站设置 - 基本设置 - 注册是否需要验证码'参数判断是否含有值 |
| `userCustomList` | [List&lt;UserCustom>](#anchor:用户自定义注册功能项-UserCustom) | 用户自定义注册功能项 |                                      |
| `allowRegisterAccountType` | List&lt;Integer>                                     | 允许注册账号类型 |                                      |
| `isAllowForeignCellPhoneRegistration` | boolean                                                     | 是否允许国外手机号注册 |                                      |
