-  **接口说明：** 在线帮助分页
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 自定义 或 [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Help>](#anchor:分页-PageView)


### 请求参数表
| 字段                | 类型     | 是否必填 | 说明   |
|:------------------|:-------|:-----|:-----|
| `page`            | Integer | 否    | 页码   |
| `helpTypeId`      | Long | 否    | 帮助分类Id |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Help>](#anchor:帮助-Help)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 帮助 `Help`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | Long          | 帮助Id    |                    |
| `name` | String        | 帮助名称      |                        |
| `userName` | String        | 用户名称      |                        |
| `helpTypeId` | Long          | 帮助分类Id    |                        |
| `content` | String        | 帮助内容    |                        |
| `times` | LocalDateTime | 发表时间    |                        |
