-  **接口说明：** 我的问题
-  **接口地址：** /user/control/questionList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Question>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |



### 分页 `PageView`
| 字段 | 类型                                       | 描述          | 备注                                   |
| :--- |:-----------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Question>](#anchor:问题-Question) | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)      | 分页索引 |                                      |
| `totalpage` | long                                     | 总页数 |                                      |
| `maxresult` | long                                     | 每页显示记录数 |                                      |
| `currentpage` | long                                     | 当前页 |                                      |
| `pagecount` | long                                     | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 问题 `Question`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | Long          | 问题Id    |                    |
| `title` | String        | 标题      |                        |
| `questionTagAssociationList` | [List&lt;QuestionTagAssociation>](#anchor:问题标签关联-QuestionTagAssociation)           | 问题标签关联集合    |                        |
| `summary` | String        | 内容摘要    |                        |
| `postTime` | LocalDateTime | 发表时间    |                        |
| `adoptionAnswerId` | Long          | 采纳的答案Id    |                        |
| `answerTotal` | Long          | 答案总数      |                        |
| `allow` | boolean       | 允许回答      |                        |
| `viewTotal` | Long          | 查看总数      |                        |
| `userName` | String        | 用户名称      |                        |
| `isStaff` | Boolean       | 是否为员工      |                        |
| `status` | Integer       | 状态      |         10.待审核 20.已发布               |
| `amount` | BigDecimal    | 悬赏金额      |                        |
| `point` | Long              | 悬赏积分      |                        |



### 问题标签关联 `QuestionTagAssociation`
| 字段         | 类型                            | 描述     | 备注                                                           |
|:-----------|:------------------------------|:-------|:-------------------------------------------------------------|
| `id`       | long                       | Id     |                    |
| `questionTagId` | long | 问题标签Id |                        |
| `questionTagName` | String | 问题标签名称 |                        |
| `questionId` | long | 问题Id |                        |
