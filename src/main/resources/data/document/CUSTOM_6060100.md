-  **接口说明：** 会员卡列表
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 无
-  **返回类型：** [PageView&lt;MembershipCard>](#anchor:分页-PageView)




### 分页 `PageView`
| 字段 | 类型                                       | 描述          | 备注                                   |
| :--- |:-----------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;MembershipCard>](#anchor:会员卡-MembershipCard) | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)      | 分页索引 |                                      |
| `totalpage` | long                                     | 总页数 |                                      |
| `maxresult` | long                                     | 每页显示记录数 |                                      |
| `currentpage` | long                                     | 当前页 |                                      |
| `pagecount` | long                                     | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |
