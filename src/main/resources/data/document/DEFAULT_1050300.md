-  **接口说明：** 粉丝列表
-  **接口地址：** /user/control/followerList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Follower>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Follower>](#anchor:粉丝-Follower)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 粉丝 `Follower`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | String          | 粉丝Id    |                    |
| `userName` | String        | 关注的用户名称      |                        |
| `friendUserName` | String        | 对方的用户名称      |                        |
| `friendAccount` | String          | 对方账号    |                        |
| `friendNickname` | String          |   对方呢称  |                        |
| `friendAvatarPath` | String          |   对方头像路径  |                        |
| `friendAvatarName` | String          |  对方头像名称   |                        |
| `addtime` | LocalDateTime | 加入时间    |                        |
