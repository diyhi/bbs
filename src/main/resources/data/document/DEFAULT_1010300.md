-  **接口说明：** 话题文件上传
-  **接口地址：** /user/control/topic/upload
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型                                | 是否必填          | 说明 |
|:-----------|:----------------------------------|:------------|:---|
| `file`      | MultipartFile                     |  是  | 上传文件接收字段  |
| `dir`      | String                            | 是 |  文件上传类型 分别为`image`、`file`、`media`  |
| `fileName`    | String                            |按需填写 |  文件名称 预签名时必须有值  |



### 返回结果
| 字段 | 类型                                  | 描述          | 备注                                    |
| :--- |:------------------------------------|:------------|:--------------------------------------|
| `error` | int                                 | 成功返回`0` 失败返回`1` |  |
| `url` | String                              | 上传图片路径      | 当字段`error`等于`0`时有值 |
| `message` | String                              | 上传返回信息      |当字段`error`等于`1`时有值|
| `title` | String                              | 旧文件名称       |当字段`error`等于`0`时有值|
