-  **接口说明：** 用户动态列表
-  **接口地址：** /user/control/userDynamicList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;UserDynamic>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型      | 是否必填 | 说明   |
|:-----------|:--------|:-----|:-----|
| `page`      | Integer | 否 | 页码   |
| `userName`      | String  | 按需填写 | 用户名称 |


### 分页 `PageView`
| 字段 | 类型                                     | 描述          | 备注                                   |
| :--- |:---------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;UserDynamic>](#anchor:用户动态-UserDynamic) | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)    | 分页索引 |                                      |
| `totalpage` | long                                   | 总页数 |                                      |
| `maxresult` | long                                   | 每页显示记录数 |                                      |
| `currentpage` | long                                   | 当前页 |                                      |
| `pagecount` | long                                   | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 用户动态 `UserDynamic`
| 字段 | 类型            | 描述      | 备注                                                                                                                                                                                                                                       |
| :--- |:--------------|:--------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `id` | String          | Id    |                                                                                                                                                                                                                                          |
| `userName` | String          | 用户名称    |                                                                                                                                                                                                                                          |
| `account` | String        | 账号    |                                                                                                                                                                                                                                          |
| `nickname` | String        | 呢称    |                                                                                                                                                                                                                                          |
| `avatarPath` | String | 头像路径  | 页面上显示200 &times; 200头像示例{{userDynamic.friendAvatarPath}}{{userDynamic.friendAvatarName}}   <br>显示100 &times; 100头像示例{{userDynamic.friendAvatarPath}}100x100/{{userDynamic.friendAvatarName}}                                             |
| `avatarName` | String        | 头像名称      |                                                                                                                                                                                                                                          |
| `postTime` |     LocalDateTime    |   发表时间    |                                                                                                                                                                                                                                          |
| `status` |     Integer    |   状态    | 10.待审核 20.已发布 110.待审核删除 120.已发布删除                                                                                                                                                                                                        |
| `module` |     Integer    |   模块    | [100.话题](#anchor:话题-100) <br>    [200.评论](#anchor:评论-200) <br>    [300.引用评论](#anchor:引用评论-300) <br>   [400.回复](#anchor:回复-400) <br>     [500.问题](#anchor:问题-500) <br>   [600.答案](#anchor:答案-600) <br> [700.答案回复](#anchor:答案回复-700) <br>  |




### 话题 `100`
