-  **接口说明：** 添加举报
-  **接口地址：** /user/control/report/add
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型      | 是否必填          | 说明                                        |备注                                    |
|:-----------|:--------|:------------|:------------------------------------------|:------------------------------------------|
| `reportTypeId`      | String        |  是  | 举报分类Id                                    ||
| `reason`    | String  |按需填写 | 理由                                        ||
| `parameterId` | String | 是 | 参数Id                                      | 话题Id；  评论Id；  评论回复Id；  问题Id；  答案Id；  答案回复Id|
| `module` | Integer  | 是 | 模块                                |10:话题  20:评论  30:评论回复  40:问题  50:答案   60:答案回复|
| `imageFile` | MultipartFile[]  | 按需填写 | 上传图片文件                                     |根据'全站设置 - 基本设置 - 举报图片允许最大上传数量'参数计算允许上传数量|
| `captchaKey` | String     | 按需填写 | 验证码编号 |
| `captchaValue`  | String     | 按需填写 |  验证码值 同一编号验证码每分钟提交次数不能超过60次,超出时验证码图片返回空字符  |




### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |
| `captchaKey` | String                                      |新验证码编号 |当`success`参数返回`false`时根据'全站设置 - 基本设置 - 举报每分钟提交超过'参数计算是否含有值|


### 错误结果 `error`
| 字段 | 类型              | 描述     |
| :--- |:----------------|:-------|
| `captchaValue` | String          | 验证码错误  | 
| `report` | String          | 举报错误   | 
| `reason` | String          | 理由错误 |
| `reportTypeId` | String          | 举报分类错误 |
| `imageFile` | String          | 上传图片错误 |
