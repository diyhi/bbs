-  **接口说明：** 账户余额
-  **接口地址：** /user/control/balance
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |



### 返回结果
| 字段 | 类型                                           | 描述      | 备注                                                      |
| :--- |:---------------------------------------------|:--------|:--------------------------------------------------------|
| `pageView` | [PageView&lt;PaymentLog>](#anchor:分页-PageView) | 积分日志分页    |                                                         |
| `deposit` | BigDecimal           | 预存款 |  |




### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;PaymentLog>](#anchor:支付日志-PaymentLog)                             | 数据展示列表 |   |

| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |

| `interfaceProduct` | Integer        | 接口产品    | -1:员工操作  0:预存款支付  1.支付宝即时到账  4.支付宝手机网站                                                                                             |
| `tradeNo` | String          |  交易号   |                                                                                                                  |

| `amount` | BigDecimal          |  金额   |         精度为12位，小数点位数为2位                                                                                                         |

| `remark` | String          |  备注   |                                                                                                                  |
