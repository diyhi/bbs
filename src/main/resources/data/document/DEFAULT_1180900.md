-  **接口说明：** 找回密码第二步表单
-  **接口地址：** /forgotPassword/step1
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)


### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 | 备注                                          |
|:-----------|:-----------|:-----|:---|:--------------------------------------------|
| `userName`      | String       | 是    | 用户名称  |                                             |


### 返回结果
| 字段 | 类型                                          | 描述                     | 备注                                                         |
| :--- |:--------------------------------------------|:-----------------------|:-----------------------------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |                                                            |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合                  |                                                            |
| `captchaKey` | String                                      | 新验证码编号                 | 当`success`参数返回`false`时有值 |
| `user` | [User](#anchor:用户-User)               | 用户                     |                                                            |


### 用户 `User`
| 字段         | 类型              | 描述    | 备注                                                                                                                                          |
|:-----------|:----------------|:------|:--------------------------------------------------------------------------------------------------------------------------------------------|
| `id`       | Long          | 用户Id  |                                                                                                                                             |
| `userName` | String          | 会员用户名  |                                                                                                                                             |
| `account`  | String          | 账号 |                                                                                                                                             |
| `issue`    | String          | 密码提示问题 |                                                                                                                                             |
| `type`     | String          | 用户类型 |                   10:本地账号密码用户 20: 手机用户 30: 邮箱用户 40:微信用户 80:其他用户                                                                                                                          |



### 错误结果 `error`
| 字段 | 类型              | 描述              |
| :--- |:----------------|:----------------|
| `account` | String          | 账号错误            |
