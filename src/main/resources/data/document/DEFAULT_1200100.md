-  **接口说明：** 用户中心
-  **接口地址：** /user/control/home
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `userName`      | String | 按需填写 | 用户名称  |



### 返回结果
| 字段 | 类型                                           | 描述      | 备注                                                      |
| :--- |:---------------------------------------------|:--------|:--------------------------------------------------------|
| `user` | [User](#anchor:用户-User)            | 用户 |  |




### 用户 `User`
| 字段                       | 类型                                                 | 描述                         | 备注       |
|:-------------------------|:---------------------------------------------------|:---------------------------|:---------|
| `id`                     | Long                                               | Id                         | 员工账号id为-1 |
| `userName`               | String                                             | 用户名称                       |          |
| `account`                | String                                             | 账号                         |          |
| `nickname`               | String                                             | 呢称                         |          |
| `issue`                  | String                                             | 密码提示问题                     |          |
| `registrationDate`       | LocalDateTime                                      | 注册日期                       |          |
| `point`                  | Long                                      | 当前积分                       |          |
| `gradeId`                | Integer                                      | 等级Id                       |          |
| `gradeName`              | String                                      | 等级名称                       |          |
| `mobile`                 | String                                      | 绑定手机                       |          |
| `realNameAuthentication` |     boolean                                  | 是否实名认证                     |          |
| `avatarPath`             | String          | 头像路径                       |     页面上显示200 &times; 200头像示例{{user.avatarPath}}{{user.avatarName}}  <br> 显示100 &times; 100头像示例{{user.avatarPath}}100x100/{{user.avatarName}} | 
| `avatarName`             | String          | 头像名称                       |                       |
| `ipAddress`              | String          | IP归属地                      | 只有中国地区简体中文语言才显示IP归属地                      |
| `userRoleNameList`       |  List&ltString>        | 话题允许查看的角色名称集合(默认角色除外)      |                        |
