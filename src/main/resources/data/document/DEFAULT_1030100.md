-  **接口说明：** 添加收藏夹
-  **接口地址：** /user/control/favorite/add
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 | 备注  |
|:-----------|:-----------|:-----|:---|:---|
| `topicId`      | Long       | 按需填写    | 话题Id  | 话题Id和问题Id必须选填一个 |
| `questionId`      | Long       | 按需填写    | 问题Id  | 话题Id和问题Id必须选填一个 |




### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |

### 错误结果 `error`
| 字段 | 类型              | 描述     |
| :--- |:----------------|:-------|
| `topicFavorite` | String          | 话题收藏错误   | 
| `questionFavorite` | String          | 问题收藏错误 |
| `favorites` | String          | 收藏夹错误 |
