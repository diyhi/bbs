-  **接口说明：** 领取红包用户分页
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;ReceiveRedEnvelope>](#anchor:分页-PageView)


### 请求参数表
| 字段                | 类型     | 是否必填 | 说明   |
|:------------------|:-------|:-----|:-----|
| `page`            | Integer | 否    | 页码   |
| `giveRedEnvelopeId`      | String | 是    | 发红包Id |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;ReceiveRedEnvelope>](#anchor:收红包-ReceiveRedEnvelope)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 收红包 `ReceiveRedEnvelope`
| 字段                 | 类型            | 描述         | 备注                                                           |
|:-------------------|:--------------|:-----------|:-------------------------------------------------------------|
| `id`               | Long          | Id         |                    |
| `receiveUserId`    | Long       | 收红包的用户Id      |                      |
| `receiveUserName`  | String        | 收红包的用户名称       |                        |
| `receiveAccount`   | String        | 收红包的账号         |                        |
| `receiveNickname`         | String        | 收红包的用户呢称         |                        |
| `receiveAvatarPath`       | String        | 收红包的用户头像路径       |     页面上显示200 &times; 200头像示例{{receiveRedEnvelope.receiveAvatarPath}}{{receiveRedEnvelope.receiveAvatarName}}  <br>显示100 &times; 100头像示例{{receiveRedEnvelope.receiveAvatarPath}}100x100/{{receiveRedEnvelope.receiveAvatarName}}                    |
| `receiveAvatarName`       | String        | 收红包的用户头像名称       |                        |
| `giveRedEnvelopeId`   | String        | 发红包Id     |    |
| `giveUserId` | Long        | 发红包的用户Id   |
| `receiveTime`         | LocalDateTime | 收取时间       |                        |
| `amount`       | BigDecimal          | 红包金额       |                        |
