-  **接口说明：** 举报列表
-  **接口地址：** /user/control/reportList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Report>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Report>](#anchor:举报-Report)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 举报 `Report`
| 字段 | 类型            | 描述      | 备注                                                                                                                                                                                                                     |
| :--- |:--------------|:--------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `id` | Long          | Id    |                                                                                                                                                                                                                        |
| `userId` | String        | 用户Id      |                                                                                                                                                                                                                        |
| `reportTypeId` | String        | 举报分类Id      |                                                                                                                                                                                                                        |
| `reportTypeName` | String        | 举报分类名称    |                                                                                                                                                                                                                        |
| `parameterId` | String        |   参数Id  | 话题Id；  评论Id；  评论回复Id；  问题Id；  答案Id；  答案回复Id                                                                                                                                                                            |
| `extraParameterId` | String        |   扩展参数Id  | 模块值10：空；  模块值20：话题Id； 模块值30：话题Id-评论Id；  模块值40：空；  模块值50：问题Id；  模块值60：问题Id-答案Id                                                                                                                                         |
| `module` | Integer              |  模块   | 10:话题  20:评论  30:评论回复  40:问题  50:答案   60:答案回复                                                                                                                                                                          |
| `reportTypeId` | String        | 举报分类Id      |                                                                                                                                                                                                                        |
| `reason` | String        | 举报理由      |                                                                                                                                                                                                                        |
| `processResult` | String        | 处理结果      |                                                                                                                                                                                                                        |
| `imageInfoList` | [ImageInfo](#anchor:图片信息-ImageInfo)          | 图片      |                                                                                                                                                                                                                        |
| `postTime` | LocalDateTime | 举报时间    |                                                                                                                                                                                                                        |
| `processCompleteTime` | LocalDateTime | 处理完成时间    | 处理状态(40:投诉失败  50:投诉成功)为处理完成状态                                                                                                                                                                                          |
| `status` | Integer | 处理状态    | 10:待处理  40:投诉失败  50:投诉成功  <br> 1010:待处理【用户删除】  1020:需补充理由【用户删除】  1030:已补充理由【用户删除】  1040:投诉失败【用户删除】  1050:投诉成功【用户删除】   <br>  100010:待处理【员工删除】  100020:需补充理由【员工删除】  100030:已补充理由【员工删除】  100040:投诉失败【员工删除】  100050:投诉成功【员工删除】 |


### 图片信息 `ImageInfo`

| `path` | String                       | 图片路径 |                    |
| `name` | String | 图片名称 |                        |
