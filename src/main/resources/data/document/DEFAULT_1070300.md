-  **接口说明：** 发红包金额分配列表
-  **接口地址：** /user/control/redEnvelopeAmountDistributionList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |
| `giveRedEnvelopeId`      | String | 是    | 发红包Id  |


### 返回结果
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `giveRedEnvelope` | [GiveRedEnvelope](#anchor:发红包-GiveRedEnvelope)                             | 发红包 |   |
| `pageView` | [PageView&lt;ReceiveRedEnvelope>](#anchor:分页-PageView)                  | 发红包金额分配分页 |                                      |



### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;ReceiveRedEnvelope>](#anchor:发红包金额分配-ReceiveRedEnvelope)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 发红包 `GiveRedEnvelope`
| 字段 | 类型            | 描述       | 备注                                                           |
| :--- |:--------------|:---------|:-------------------------------------------------------------|
| `id` | String          | 发红包Id    |                    |
| `userId` | Long        | 发红包的用户Id |                        |
| `type` | Integer        | 类型       |       10.个人定向红包、20.公共随机红包(随机金额)、30.公共定额红包(固定金额)                 |
| `totalAmount` | BigDecimal          | 总金额      |                        |
| `singleAmount` | BigDecimal          | 单个红包金额      |          type=30时有值              |
| `giveQuantity` | Integer          | 发放数量   |                        |
| `bindTopicId` | Long          | 绑定话题Id   |           -1表示空值               |
| `bindTopicTitle` | String | 绑定话题标题     |                        |
| `remainingQuantity` | Integer | 剩余数量     |                        |
| `giveTime` | LocalDateTime | 发放时间     |                        |
