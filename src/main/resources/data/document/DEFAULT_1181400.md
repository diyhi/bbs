-  **接口说明：** 积分
-  **接口地址：** /user/control/point
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |



### 返回结果
| 字段 | 类型                                           | 描述      | 备注                                                      |
| :--- |:---------------------------------------------|:--------|:--------------------------------------------------------|
| `pageView` | [PageView&lt;PointLog>](#anchor:分页-PageView) | 积分日志分页    |                                                         |
| `user` | [User](#anchor:用户-User)            | 用户 |  |
| `rewardPointInfo` | [RewardPointInfo](#anchor:奖励积分信息-RewardPointInfo)     | 奖励积分信息 |                                                         |




### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;PointLog>](#anchor:积分日志-PointLog)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |




### 积分日志 `PointLog`
| 字段                  | 类型            | 描述      | 备注                                                                                                                   |
|:--------------------|:--------------|:--------|:---------------------------------------------------------------------------------------------------------------------|
| `id`                | String          | 话题Id    |                                                                                                                      |
| `module`            | Integer        | 模块      | 100.话题  200.评论  300.回复 400.积分解锁话题隐藏内容 500.会员卡订单支付 600.账户充值 700.问题 800.答案 900.答案回复 <br> 1000.悬赏积分 1100.采纳答案 1200.调整赏金 |
| `parameterId`       | Long          | 参数Id    | 话题Id ,评论Id,回复Id 问题Id ,答案Id,答案回复Id                                                                                    |
| `operationUserType` | Integer        | 操作用户类型    | 0:系统  1: 员工  2:会员                                                                                                    |
| `operationUserName` | String          |  操作用户名称   |                                                                                                                      |
| `userName`          | String          |  用户名称   |                                                                                                                      |
| `pointState`        | int          |  积分状态   | 1:账户存入  2:账户支出                                                                                                       |
| `point`             | Long          |  积分   |                                                                                                                      |
| `times`             | LocalDateTime          |  时间   |                                                                                                                      |
| `remark`            | String          |  备注   |                                                                                                                      |
