-  **接口说明：** 答案分页
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 自定义 或 [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Answer>](#anchor:分页-PageView)


### 请求参数表
| 字段                | 类型     | 是否必填 | 说明   |
|:------------------|:-------|:-----|:-----|
| `page`            | Integer | 否    | 页码   |
| `questionId`      | Long | 是    | 问题Id |
| `answerId`       | Long | 否 | 答案Id |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Answer>](#anchor:答案-Answer)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 答案 `Answer`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | Long          | 评论Id    |                    |
| `isStaff` | boolean       | 是否为员工      |                      |
| `userName` | String        | 用户名称      |                        |
| `account` | String        | 账号      |                        |
| `nickname` | String        | 呢称      |                        |
| `userInfoStatus` | Integer        | 用户信息状态      |       -30.账号已注销(不显示数据) -20.账号已逻辑删除(不显示数据) -10.账号已禁用(不显示数据)  0.正常 10.账号已禁用(显示数据) 20.账号已逻辑删除(显示数据) |
| `userRoleNameList` | List&ltString>        | 用户角色名称集合     |
| `avatarPath` | String        | 头像路径      |     页面上显示200 &times; 200头像示例{{comment.avatarPath}}{{comment.avatarName}}  <br>显示100 &times; 100头像示例{{comment.avatarPath}}100x100/{{comment.avatarName}}                    |
| `avatarName` | String        | 头像名称      |                        |
| `questionId` | Long          | 问题Id    |                        |
| `content` | String        | 评论内容    |                        |
| `postTime` | LocalDateTime | 评论时间    |                        |
| `totalReply` | Integer          | 总回复数      |                        |
| `ipAddress` | String        | IP归属地      |                        |
| `answerReplyList` | [List&lt;AnswerReply>](#anchor:回复-AnswerReply)           | 回复集合    |                        |

| `avatarName` | String        | 头像名称      |                        |
| `questionId` | Long                       | 问题Id |                    |
| `content` | String        | 回复内容    |                        |
| `answerId` | Long                       | 答案Id |                    |
