-  **接口说明：** 根据账号或呢称查询用户
-  **接口地址：** /user/control/queryUser
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [User](#anchor:用户-User)



### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 |备注                                    |
|:-----------|:-----------|:-----|:---|:---|
| `keyword`      | String       | 是 | 关键字  ||




### 用户 `User`
| 字段           | 类型                                          | 描述   | 备注                                    |
|:-------------|:--------------------------------------------|:-----|:--------------------------------------|
| `id`         | Long                                        | Id   |  |
| `userName`   | String | 用户名称 |  |
| `account`    | String                                      | 账号   ||
| `nickname`   | String                                      | 呢称   ||
| `state`      | Integer                                      | 用户状态 ||
| `point`      | Long                                      | 当前积分 ||
| `gradeId`    | Integer                                      |  等级Id    ||
| `gradeName`  | String                                      |   等级名称   ||
| `avatarPath` | String                                      |   头像路径   |页面上显示200 &times; 200头像示例{{systemUser.avatarPath}}{{systemUser.avatarName}}  <br>显示100 &times; 100头像示例{{systemUser.avatarPath}}100x100/{{systemUser.avatarName}} |
| `avatarName` | String                                      |   头像名称   ||
