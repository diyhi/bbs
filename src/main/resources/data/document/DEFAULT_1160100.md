-  **接口说明：** 付款表单
-  **接口地址：** /user/control/payment
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 | 备注          |
|:-----------|:-----------|:-----|:---|:---|
| `paymentModule`      | Integer    | 是    | 支付模块    | 5.用户充值  | 


### 返回结果
| 字段 | 类型                                          | 描述      | 备注          |
| :--- |:--------------------------------------------|:--------|:------------|
| `onlinePaymentInterfaceList` | [List&lt;OnlinePaymentInterface>](#anchor:在线支付接口-OnlinePaymentInterface)    | 所有支付接口    |             |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 | 含有错误时才出现本字段 |




### 在线支付接口 `OnlinePaymentInterface`
| 字段                 | 类型                                  | 描述         | 备注                                                                                                                                                          |
|:-------------------|:------------------------------------|:-----------|:-----------------------------------------------------------------------|
| `id`               | Integer                              | Id         ||
| `name`             | String                              | 名称      |                                                                                                                                                             |
| `interfaceProduct` | Integer                              | 接口产品         |        1.支付宝即时到账  3. 支付宝手机网站                                                                                                                                                     |
| `bankList`         | [List&lt;Bank>](#anchor:银行-Bank)   | 支持银行集合    |                                                                |


### 银行 `Bank`
| 字段         | 类型                                  | 描述                                   | 备注                                                                                                                                                          |
|:-----------|:------------------------------------|:-------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `code`     | String                             | 银行简码                                 ||
| `name`     | String                              | 银行名称                                 |                                                                                                                                                             |
| `icon`     | String                             | 银行图标                                 |                                                                                                                                                    |
| `selected` | boolean   |    选中 true:选中  false:未选中             ||


### 错误结果 `error`
| 字段 | 类型              | 描述   |
| :--- |:----------------|:-----|
