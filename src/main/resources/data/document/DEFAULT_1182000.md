-  **接口说明：** 手机绑定
-  **接口地址：** /user/control/phoneBinding
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 |备注                                    |
|:-----------|:-----------|:-----|:---|:---|
| `countryCode`    | String     | 按需填写 | 区号   |                                             |
| `mobile`    | String     | 是 | 手机号   |                                             |
| `smsCode`    | String     | 是 | 手机验证码   |                                             |






### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |
| `captchaKey` | String                                      |新验证码编号 |当`success`参数返回`false`含有值|




### 错误结果 `error`
| 字段 | 类型              | 描述     |
| :--- |:----------------|:-------|
| `captchaValue` | String          | 验证码错误  | 
| `mobile` | String          | 手机号错误 |
| `smsCode` | String          | 手机验证码错误 |
