-  **接口说明：** 我的答案
-  **接口地址：** /user/control/answerList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Answer>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |



### 分页 `PageView`
| 字段 | 类型                                     | 描述          | 备注                                   |
| :--- |:---------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Answer>](#anchor:答案-Answer) | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)    | 分页索引 |                                      |
| `totalpage` | long                                   | 总页数 |                                      |
| `maxresult` | long                                   | 每页显示记录数 |                                      |
| `currentpage` | long                                   | 当前页 |                                      |
| `pagecount` | long                                   | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 答案 `Answer`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | Long          | 答案Id    |                    |
| `questionId` | Long          | 问题Id    |                        |
| `questionTitle` | String        | 问题标题    |                        |
| `content` | String        | 答案内容    |                        |
| `postTime` | LocalDateTime | 回答时间    |                        |
| `lastUpdateTime` | LocalDateTime | 最后修改时间  |                        |
| `userName` | String        | 用户名称      |                        |
| `isStaff` | Boolean       | 是否为员工      |                        |
| `status` | Integer        | 状态      |         10.待审核 20.已发布               |
