-  **接口说明：** 第三方登录重定向
-  **接口地址：** /thirdParty/loginRedirect
-  **请求方式：** 不限
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)


### 请求参数表
| 字段         | 类型         | 是否必填 | 说明                                                 |
|:-----------|:-----------|:-----|:---------------------------------------------------|
| `code`      | String       | 是    | 微信公众号code | 
| `state`      | String       | 是    | 微信公众号state | 



### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |
| `userAuthorization` | String                                      |用户授权 |当`success`参数返回`true`时有值|


### 错误结果 `error`
| 字段 | 类型              | 描述      |
| :--- |:----------------|:--------|
| `state` | String          | 自定义参数错误   | 
| `weiXinUserInfo` | String          | 微信用户信息错误 | 
| `weChatConfig` | String          | 微信配置信息错误    | 
| `otherUserInfo` | String          | 其他开放平台用户信息错误   | 
| `otherConfig` | String          | 其他开放平台配置信息错误   |
