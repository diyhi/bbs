-  **接口说明：** 文件下载
-  **接口地址：** /fileDownload
-  **请求方式：** 不限
-  **Ajax请求头：** 协议头：X-Requested-With  值：XMLHttpRequest
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** `HTTP 302` 或 [Map&lt;String,Object>](#anchor:Ajax请求返回结果)
-  **接口说明：** [根据请求头自动切换：Ajax 请求返回JSON信息，普通请求直接执行 302 重定向。](#anchor:接口说明)





### 请求参数表
| 字段         | 类型                                 | 是否必填          | 说明                                       |
|:-----------|:-----------------------------------|:------------|:-----------------------------------------|
| `jump`      | String                             |  是  | 重定向参数                                    |


### Ajax请求返回结果
| 字段 | 类型                                          | 描述                     | 备注                                    |
| :--- |:--------------------------------------------|:-----------------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |                          |
| `redirect` | String                                      | 重定向地址                  |当`success`参数返回`true`时有值|


### 错误结果 `error`
| 字段 | 类型              | 描述       |
| :--- |:----------------|:---------|
| `message` | String          | 错误信息     | 

### 接口说明
| 状态 | 返回内容                                          | 说明                     | 
| :--- |:--------------------------------------------|:-----------------------|
| 普通请求 | 无 (302 Redirect)                                     | 浏览器直接下载文件| 
| Ajax 请求 | {"success":true, "redirect":"..."}                                      | 前端需自行处理 redirect 字段进行跳转                  |
