-  **接口说明：** 恢复微信浏览器会话
-  **接口地址：** /recoverWeChatBrowserSession
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)


### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 | 备注                                          |
|:-----------|:-----------|:-----|:---|:--------------------------------------------|
| `code`      | String       | 是    | 换取access_token的票据  |       code作为换取access_token的票据，每次用户授权带上的code将不一样，code只能使用一次，5分钟未被使用自动过期                                      |



### 返回结果
| 字段 | 类型      | 描述                     | 备注                                    |
| :--- |:--------|:-----------------------|:--------------------------------------|
| `accessToken` | String | 访问令牌 |  |
| `refreshToken` | String  | 刷新令牌     |  |
| `systemUser` | [AccessUser](#anchor:登录用户-AccessUser)  | 登录用户     |  |
| `openId` | String  | 用户的唯一标识     |  |
