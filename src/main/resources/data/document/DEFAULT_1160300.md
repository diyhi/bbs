-  **接口说明：** 付款
-  **接口地址：** /user/control/payment
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 | 备注                                        |
|:-----------|:-----------|:-----|:---|:---|
| `paymentModule`      | Integer    | 是    | 支付模块    | 5.用户充值  | 
| `rechargeAmount`      | BigDecimal   |  是    | 充值金额 |   | 
| `paymentBank`      | String    |   是   | 支付银行 |  由 接口产品_银行简码 组成 | 


### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                        |
| :--- |:--------------------------------------------|:------------|:------------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |                                           |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |                                           |
| `callbackData` |           String                              |  回调数据| 当`success`参数返回`true`时有值，前端执行后跳转到第三方支付机构页面 |


### 错误结果 `error`
| 字段 | 类型              | 描述      |
| :--- |:----------------|:--------|
| `message` | String          | 错误信息    | 
| `rechargeAmount` | String          | 充值金额错误 | 
| `paymentBank` | String          | 支付银行错误 |
