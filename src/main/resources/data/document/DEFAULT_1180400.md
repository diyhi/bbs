-  **接口说明：** 会员登录表单
-  **接口地址：** /login
-  **请求方式：** GET
-  **请求参数：** 无
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)





### 返回结果
| 字段 | 类型                                       | 描述      | 备注                                                      |
| :--- |:-----------------------------------------|:--------|:--------------------------------------------------------|
| `formCaptcha` | [FormCaptcha](#anchor:表单验证码-FormCaptcha) | 表单验证码    |                                                         |
| `allowRegisterAccountType` | List&lt;Integer>                         | 允许注册账号类型 | 10.本地账号密码用户 20.手机用户 30.邮箱用户 40.微信用户 80.其他开放平台用户 |
| `isAllowForeignCellPhoneRegistration` | boolean                         | 是否允许国外手机号注册 |                                                         |




### 表单验证码 `FormCaptcha`
| 字段                 | 类型                                  | 描述         | 备注                                                                                                                                                          |
|:-------------------|:------------------------------------|:-----------|:-----------------------------------------------------------------------|
| `showCaptcha`               | boolean                              | 是否显示验证码         ||
| `captchaKey`             | String                              | 验证码编号      |                                                                                                                                                             |
