-  **接口说明：** 获取绑定短信验证码
-  **接口地址：** /user/control/getSmsCode
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型                                 | 是否必填          | 说明                                       | 备注                                                       |
|:-----------|:-----------------------------------|:------------|:-----------------------------------------|:---------------------------------------------------------|
| `module`      | Integer                             |  是  | 模块                                    | 1.绑定手机  2.更换绑定手机第一步  3.更换绑定手机第二步 |
| `countryCode`      | String                             |  按需填写  | 区号                                   | +86区号可以省略                                                |
| `mobile`      | String                             |  是  | 手机号                                   |                                                          |
| `captchaKey`      | String                             |  是  | 验证码编号                                    |                                                          |
| `captchaValue`      | String                             |  是  | 验证码值                                    |                                                          |





### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |
| `captchaKey` | String                                      |新验证码编号 |当`success`参数返回`false`时有值|


### 错误结果 `error`
| 字段 | 类型              | 描述      |
| :--- |:----------------|:--------|
| `captchaValue` | String          | 验证码错误   | 
| `smsCode` | String          | 短信验证码错误 | 
| `message` | String          | 消息错误    | 
| `mobile` | String          | 手机号错误   |
