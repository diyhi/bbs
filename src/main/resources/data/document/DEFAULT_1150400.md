-  **接口说明：** 基本信息
-  **接口地址：** /baseInfo
-  **请求方式：** GET
-  **请求参数：** 无
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 返回结果
| 字段 | 类型                                  | 描述      | 备注                                     |
| :--- |:------------------------------------|:--------|:---------------------------------------|
| `title` | String                              | 站点名称    |
| `keywords` | String                              | 站点关键词   |                                        |
| `description` | String                              | 站点描述    |                                        |
| `supportAccessDevice` | Integer                             | 支持访问设备  | 1.自动识别终端 2.电脑端 3.移动端                   |
| `baseURL` | String                              | 系统路径    |                                        |
| `contextPath` | String                              | 系统虚拟目录  |                                        |
| `systemUser` | [AccessUser](#anchor:用户-AccessUser) | 登录用户    |                                        |
| `baseURI` | String                              | 系统资源标识符 |                                        |
| `fileStorageSystem` |         int                            | 文件存储系统  | 0.本地系统 10.SeaweedFS 20.MinIO 30.阿里云OSS |
| `languageSwitching` | Map<String,String>                  | 多语言切换   | key:语言代号 value:语言名称                    |
| `weixin_oa_appid` | String                              | 微信公众号appid        |                                        |
