-  **接口说明：** 投票主题明细
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [VoteTheme](#anchor:投票主题-VoteTheme)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `voteThemeId`      | String | 是    | 投票主题Id  |




### 投票主题 `VoteTheme`
| 字段 | 类型                                  | 描述         | 备注                                                                                                                                                                                                                                                                                                                                                       |
| :--- |:------------------------------------|:-----------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `id` | String                              | 投票主题Id    |                    |
| `title` | String                              | 投票标题      |                        |
| `module` | Integer                             | 模块      |      10:话题  20:问答                  |
| `sourceParameterTitle` | String                              | 来源参数Id    |      话题Id、问题Id                  |
| `maxChoice` | Integer                             | 最大可选数    |         1:单选  -1:不限制/多选               |
| `createDate` | LocalDateTime                       | 创建时间    |                        |
| `endDate` | LocalDateTime                       | 截止日期    |                        |
| `userName` | String                              | 发起投票用户名称      |                        |
| `status` | Integer                             | 状态      |       10:未开始  20.进行中  30.已结束                 |	
| `isStaff` | Boolean                             | 是否为员工      |         true:员工  false:会员               |	
| `voteOptionList` | [List&lt;VoteOption>](#anchor:投票选项-VoteOption) | 投票选项集合      |                     |	




### 投票选项 `VoteOption`
| 字段 | 类型            | 描述       | 备注                                                                                                                                                                                                                                                                                                                                                       |
| :--- |:--------------|:---------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `id` | String          | Id       |                    |
| `text` | String        | 选项内容     |                        |
| `voteThemeId` | String        | 投票主题Id     |                        |
| `sort` | Integer          | 排序   |                        |
| `selected` | boolean          | 是否选中   |                        |
| `totalVotes` | String        | 总票数     |                        |
