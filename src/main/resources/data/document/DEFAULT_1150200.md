-  **接口说明：** 校验验证码
-  **接口地址：** /checkCaptcha
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [CaptchaVerification](#anchor:返回结果-CaptchaVerification)



### 请求参数表
| 字段         | 类型                                 | 是否必填          | 说明                                       |
|:-----------|:-----------------------------------|:------------|:-----------------------------------------|
| `captchaKey`      | String                             |  是  | 验证码编号                                    |
| `captchaValue`      | String                             | 是 | 验证码值 同一编号验证码每分钟提交次数不能超过60次,超出时验证码图片返回空字符                                    |


### 返回结果 `CaptchaVerification`
| 字段 | 类型       | 描述     | 备注                                    |
| :--- |:---------|:-------|:--------------------------------------|
| `success` | Boolean | 是否验证成功 ||
