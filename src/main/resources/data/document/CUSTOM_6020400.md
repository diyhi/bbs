-  **接口说明：** 问答标签列表
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 无
-  **返回类型：** [List&lt;QuestionTag>](#anchor:问题标签-QuestionTag)




### 问题标签 `QuestionTag`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | Long          | Id    |                    |
| `name` | String        | 标签名称      |                        |
| `parentId` | Long        | 所属父标签Id      |       0为根节点                 |
| `childNodeNumber` | Integer        | 子节点数量      |                        |
| `sort` | Integer        | 排序      |       |
| `image` | String        | 图片      |                        |
| `allowRoleViewList` | List&ltString>        | 标签下的话题允许查看的角色名称集合(默认角色除外)     |
| `childTag` | [List&lt;QuestionTag>](#anchor:问题标签-QuestionTag)           | 子标签    |                        |
