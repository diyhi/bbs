-  **接口说明：** 支付回调通知
-  **接口地址：** /notify/{interfaceProduct}
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** String
-  **返回结果：** `success` 或 `fail`


### 请求参数表
| 字段         | 类型         | 是否必填 | 说明                                          |备注                                        |
|:-----------|:-----------|:-----|:--------------------------------------------|:--------------------------------------------|

| `out_trade_no`      | String     |   是   | 商户订单号  |  | 
| `trade_no`      | String      |   是   | 支付宝交易号 |  | 
| `trade_status`      | String   |   是   | 交易状态 |  |
