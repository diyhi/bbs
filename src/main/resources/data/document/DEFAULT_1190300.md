-  **接口说明：** 添加私信表单
-  **接口地址：** /user/control/addPrivateMessage
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)


### 请求参数表
| 字段         | 类型     | 是否必填          | 说明 |
|:-----------|:-------|:------------|:---|
| `friendUserName`      | String |  是  | 对方用户名称  |



### 返回结果
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `allowSendPrivateMessage` | boolean                       | 是否允许发私信 |                    |
| `formCaptcha` | [FormCaptcha](#anchor:表单验证码-FormCaptcha) | 表单验证码 |                        |



### 表单验证码 `FormCaptcha`
| 字段 | 类型                                 | 描述          | 备注                                   |
| :--- |:-----------------------------------|:------------|:-------------------------------------|
| `showCaptcha` | boolean                            | 是否显示验证码 | 根据'全站设置 - 基本设置 - 发表私信每分钟提交超过'参数计算是否含有值  |
| `captchaKey` | String                             | 验证码编号 |                                      |
