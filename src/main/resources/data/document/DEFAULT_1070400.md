-  **接口说明：** 收红包列表
-  **接口地址：** /user/control/receiveRedEnvelopeList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;ReceiveRedEnvelope>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;ReceiveRedEnvelope>](#anchor:收红包-ReceiveRedEnvelope)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 收红包 `ReceiveRedEnvelope`
| 字段                  | 类型            | 描述         | 备注                                                                                                                                                                                                                           |
|:--------------------|:--------------|:-----------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `id`                | String        | Id         ||
| `giveUserId`        | Long          | 发红包的用户Id   ||

| `giveAvatarName`     | String          | 发红包的用户头像名称   |                                                                                                                                                                                                                              |

| `receiveTime`       | LocalDateTime | 收取时间       |                                                                                                                                                                                                                              |
| `amount`            | BigDecimal    | 红包金额       |    |
