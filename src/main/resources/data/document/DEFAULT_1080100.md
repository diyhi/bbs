-  **接口说明：** 添加在线留言
-  **接口地址：** /feedback/control/add
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填          | 说明 |
|:-----------|:-----------|:------------|:---|
| `name`      | String       |  是  | 名称  |
| `contact`      | String       |  是  | 联系方式  |
| `content`    | String     |是 |  内容  |
| `captchaKey` | String     | 是 | 验证码编号 |
| `captchaValue`  | String     | 是 |  验证码值 同一编号验证码每分钟提交次数不能超过60次,超出时验证码图片返回空字符  |




### 返回结果
| 字段 | 类型                                          | 描述          | 备注                       |
| :--- |:--------------------------------------------|:------------|:-------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |                          |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |                          |
| `captchaKey` | String                                      |新验证码编号 | 当`success`参数返回`false`时有值 |


### 错误结果 `error`
| 字段 | 类型              | 描述     |
| :--- |:----------------|:-------|
| `captchaValue` | String          | 验证码错误  | 
| `feedback` | String          | 在线留言错误   | 
| `name` | String          | 在线留言名称错误 |
| `contact` | String          | 联系方式错误 |
| `content` | String          | 在线留言内容错误 |
