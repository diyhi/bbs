-  **接口说明：** 更新头像
-  **接口地址：** /user/control/updateAvatar
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)


### 请求参数表
| 字段        | 类型         | 是否必填 | 说明 | 备注                                           |
|:----------|:-----------|:-----|:---|:---------------------------------------------|
| `imgFile` | MultipartFile       | 是    | 头像 | 上传后生成图片尺寸为 200 &times; 200 和 100 &times; 100 |
| `width`   | Integer       | 是    | 图片宽度 |                                              |
| `height`  | Integer       | 是    | 图片高度 |                                              |
| `x`       | Integer       | 是    | X轴 |                                              |
| `y`       | Integer       | 是    | Y轴 |                                              |




### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |



### 错误结果 `error`
| 字段        | 类型              | 描述     |
|:----------|:----------------|:-------|
| `imgFile` | String          | 图片错误   | 
| `width`   | String          | 图片宽度错误 | 
| `height`  | String          | 图片高度错误     |
| `x`       | String          | X轴错误   |
| `y`    | String          | Y轴错误 |
