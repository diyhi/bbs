-  **接口说明：** 动态播放列表
-  **接口地址：** /dynamicPlaylist
-  **请求方式：** 不限
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** byte[]
-  **返回结果：** HLS 播放索引文件m3u8二进制文件流



### 请求参数表
| 字段         | 类型     | 是否必填          | 说明                                       | 备注                                   |
|:-----------|:-------|:------------|:-----------------------------------------|:-----------------------------------------|
| `jump`      | String |  是  | 重定向参数                                    | |



### 返回状态码
| 状态码     | 返回内容                                          | 说明                     | 
|:--------|:--------------------------------------------|:-----------------------|
| 200 OK    | 二进制文件流                                     | 返回 video.m3u8 文件内容| 
| 204 No Content | 无                                     | 请求参数 jump 为空，不返回任何数据                  |
| 404 Not Found    | 无                                     | 视频不存在、转码未完成或 m3u8 物理文件丢失|
