-  **接口说明：** 添加话题表单
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 无
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果) 



### 返回结果
| 字段 | 类型                                           | 描述          | 备注                                                           |
| :--- |:---------------------------------------------|:------------|:-------------------------------------------------------------|
| `availableTag` | String                                       | 话题编辑器允许使用的标签 | List&lt;String> 类型的JSON格式，配合 KindEditor 编辑器使用                   |
| `captchaKey` | String                                       | 验证码编号 | 根据'全站设置 - 基本设置 - 发表话题每分钟提交超过'参数计算是否含有值                       |
| `allowTopic` | boolean                                      |是否允许提交话题 |                                                              |
| `deposit` | BigDecimal                                   | 用户共有预存款 |                                                              |
| `giveRedEnvelopeAmountMin` | BigDecimal                                   | 发红包金额下限 |                                                              |
| `allowTagIdList` | List<String>                                 | 有提交权限的标签Id |                                                              |
| `giveRedEnvelopeAmountMax` | BigDecimal                                   | 发红包金额上限 | 空为无限制                                                        |
| `maxVoteOptions` | Integer                                      | 发起投票选项数量 | 空值为不限制； 0为不允许发起投票                                            |
| `userGradeList` | [List&lt;UserGrade>](#anchor:用户等级-UserGrade) | 用户等级集合 |                                                              |
| `fileSystem` | int                                          | 文件系统 | 0.本地 10.SeaweedFS 20.MinIO 30.阿里云OSS                         |
| `supportEditor` | int                                       | 支持编辑器 | 10.仅富文本编辑器 20.仅Markdown编辑器  30.富文本编辑器优先 40.Markdown编辑器优先     |
