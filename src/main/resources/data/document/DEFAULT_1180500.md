-  **接口说明：** 会员登录
-  **接口地址：** /login
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填 | 说明 |备注                                    |
|:-----------|:-----------|:-----|:---|:---|
| `account`      | String       | 按需填写 | 账号  ||
| `type`    | Integer     | 是    |  用户类型  | 10:本地账号密码用户 20: 手机用户 30: 邮箱用户 40:微信用户 80:其他用户|
| `countryCode`    | String     | 按需填写 | 区号   |                                             |
| `mobile`    | String     | 按需填写 | 手机号   |                                             |
| `email`    | String     | 按需填写 |  邮箱  |                                             |
| `password`    | String     | 是  |  密码   | 必须执行sha256一次                                |
| `captchaKey` | String     | 按需填写 | 验证码编号 ||
| `captchaValue`  | String     | 按需填写 |  验证码值 同一编号验证码每分钟提交次数不能超过60次,超出时验证码图片返回空字符  ||
| `thirdPartyOpenId`    | String     | 按需填写 | 第三方用户获取唯一标识   |      例如微信公众号openid                                       |
| `jumpUrl`    | String     | 否 |   提交成功后跳转路径,不填写默认跳转到首页 |       参数必须经过encodeBase64URL编码后才能提交                                      |




### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |
| `captchaKey` | String                                      |新验证码编号 |当`success`参数返回`false`时根据'全站设置 - 基本设置 - 评论每分钟提交超过'参数计算是否含有值|
| `jumpUrl` | String                                      | 提交成功后跳转路径,不填写默认跳转到首页|                                                            |
| `systemUser` | [AccessUser](#anchor:登录用户-AccessUser)            |登录用户 |                                                            |
| `accessToken` | String                                      | 访问令牌|                                                            |
| `refreshToken` | String                                      | 刷新令牌|                                                            |


### 登录用户 `AccessUser`
| 字段         | 类型              | 描述     | 备注                                                                                                                                          |
|:-----------|:----------------|:-------|:--------------------------------------------------------------------------------------------------------------------------------------------|
| `userId`   | Long          | 用户Id   |                                                                                                                                             |
| `userName` | String          | 会员用户名   |                                                                                                                                             |
| `account`  | String          | 账号 |                                                                                                                                             |
| `nickname` | String          | 呢称 |                                                                                                                                             |
| `avatarPath`  | String          | 头像路径 | 页面上显示200 &times; 200头像示例{{systemUser.avatarPath}}{{systemUser.avatarName}}  <br>显示100 &times; 100头像示例{{systemUser.avatarPath}}100x100/{{systemUser.avatarName}} |
| `avatarName`  | String          | 头像名称 |                                                                                                                                             | 
| `loginInterface`  | String          | 登录接口 | 0.本地 10.微信 50.其他开放平台                                                                                                                        |




### 错误结果 `error`
| 字段 | 类型              | 描述     |
| :--- |:----------------|:-------|
| `captchaValue` | String          | 验证码错误  | 
| `type` | String          | 用户类型错误          |
| `password` | String          | 密码错误            |
| `account` | String          | 账号错误            | 
| `mobile` | String          | 手机号错误           |
| `email` | String          | 邮箱错误            |
| `login` | String          | 登录错误            |
