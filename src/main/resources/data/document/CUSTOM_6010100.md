-  **接口说明：** 话题分页
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** 自定义 或 [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;Topic>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |
| `tagId`      | Long | 否 | 标签Id  |



### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;Topic>](#anchor:话题-Topic)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 话题 `Topic`
| 字段 | 类型            | 描述      | 备注                                                           |
| :--- |:--------------|:--------|:-------------------------------------------------------------|
| `id` | Long          | 话题Id    |                    |
| `title` | String        | 标题      |                        |
| `tagId` | Long          | 标签Id    |                        |
| `tagName` | String        | 标签名称    |                        |
| `summary` | String        | 内容摘要    |    不含隐藏标签内的内容                    |
| `imageInfoList` | [List&lt;ImageInfo>](#anchor:图片信息-ImageInfo)           | 图片集合    |         不含隐藏标签内的图片               |
| `mediaInfoList` | [List&lt;MediaInfo>](#anchor:媒体文件信息-MediaInfo)           | 媒体文件信息集合    |      不含隐藏标签内的媒体文件                  |
| `postTime` | LocalDateTime | 发表时间    |                        |
| `lastReplyTime` | LocalDateTime | 最后回复时间    |                        |
| `commentTotal` | Long          | 评论总数      |                        |
| `allow` | boolean       | 允许评论      |                        |
| `viewTotal` | Long          | 查看总数      |                        |
| `userName` | String        | 用户名称      |                        |
| `account` | String        | 账号      |                        |
| `userRoleNameList` | List&ltString>        | 用户角色名称集合     |                                                                                                                                                                                                                                                                                                                                                          |
| `allowRoleViewList` | List&ltString>        | 话题允许查看的角色名称集合(默认角色除外)     |                                                                                                                                                                                                                                                                                                                                                          |
| `hideTagTypeMap` | Map&ltInteger,Boolean>       | 内容含有隐藏标签类型      |       key: 10.输入密码可见  20.评论话题可见  30.达到等级可见 40.积分购买可见 50.余额购买可见  value:当前用户是否已对隐藏内容解锁                 |
| `nickname` | String        | 呢称      |                        |
| `userInfoStatus` | Integer        | 用户信息状态      |       -30.账号已注销(不显示数据) -20.账号已逻辑删除(不显示数据) -10.账号已禁用(不显示数据)  0.正常 10.账号已禁用(显示数据) 20.账号已逻辑删除(显示数据)                 |
| `avatarPath` | String        | 头像路径      |     页面上显示200 &times; 200头像示例{{topic.avatarPath}}{{topic.avatarName}}  <br>显示100 &times; 100头像示例{{topic.avatarPath}}100x100/{{topic.avatarName}}                    |
| `avatarName` | String        | 头像名称      |                        |
| `giveRedEnvelopeId` | String        | 发红包Id      |                        |
| `sort` | Integer       | 排序      |            默认为0            |
| `essence` | Boolean        | 是否为精华      |                        |
