-  **接口说明：** 搜索
-  **接口地址：** /search
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;SearchResult>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |
| `keyword`      | String | 是    | 关键词  |



### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;SearchResult>](#anchor:搜索-SearchResult)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 搜索 `SearchResult`
| 字段 | 类型            | 描述      | 备注                                                                                                                                                                                                                     |
| :--- |:--------------|:--------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `indexModule` | Integer          | 索引模块    |      10:话题 20：问题                                                                                                                                                                                                                   |
| `topic` | [Topic](#anchor:话题-Topic)             | 话题      |                                                                                                                                                                                                                        |
| `question` | [Question](#anchor:问题-Question)        | 问题      |                                                                                                                                                                                                                        |



### 话题 `Topic`
| 字段 | 类型            | 描述       | 备注                                                                                                                                       |
| :--- |:--------------|:---------|:-----------------------------------------------------------------------------------------------------------------------------------------|
| `id` | Long          | 话题Id     |                                                                                                                                          |
| `title` | String        | 标题       | 本字段高亮字符用 &lt;B>&lt;/B> 标签包裹                                                                                                                |
| `tagId` | Long          | 标签Id     |                                                                                                                                          |
| `tagName` | String        | 标签名称     |                                                                                                                                          |
| `content` | String        | 话题内容       | 本字段高亮字符用 &lt;B>&lt;/B> 标签包裹                                                                                                                |
| `summary` | String        | 内容摘要     | 内容摘要(不含隐藏标签内的内容)                                                                                                                         |
| `postTime` | LocalDateTime | 发表时间     |                                                                                                                                          |
| `lastReplyTime` | LocalDateTime | 最后回复时间     |                                                                                                                                          |
| `commentTotal` | Long          | 评论总数     |                                                                                                                                          |
| `allow` | boolean       | 允许评论     |                                                                                                                                          |
| `viewTotal` | Long          | 查看总数     |                                                                                                                                          |
| `userName` | String        | 用户名称     |                                                                                                                                          |
| `userRoleNameList` | List&ltString>        | 用户角色名称集合     |                                                                                                                                          |
| `allowRoleViewList` | List&ltString>        | 话题允许查看的角色名称集合(默认角色除外)     |                                                                                                                                          |
| `nickname` | String        | 呢称     |                                                                                                                                          |
| `avatarPath` | String        | 头像路径     | 页面上显示200 &times; 200头像示例{{topic.avatarPath}}{{topic.avatarName}}  <br> 显示100 &times; 100头像示例{{topic.avatarPath}}100x100/{{topic.avatarName}} |
| `avatarName` | String        | 头像名称     |                                                                                                                                          |
| `isStaff` | Boolean       | 是否为员工    |                                                                                                                                          |



### 问题 `Question`
| 字段 | 类型            | 描述      | 备注                          |
| :--- |:--------------|:--------|:----------------------------|
| `id` | Long          | 问题Id    |                             |
| `title` | String        | 标题      | 本字段高亮字符用 &lt;B>&lt;/B> 标签包裹 |
| `content` | String        | 问题内容       | 本字段高亮字符用 &lt;B>&lt;/B> 标签包裹   |
| `summary` | String        | 内容摘要    |                             |
| `postTime` | LocalDateTime | 发表时间    |                             |
| `lastAnswerTime` | LocalDateTime | 最后回答时间    |                             |
| `answerTotal` | Long          | 答案总数      |                             |
| `allow` | boolean       | 允许回答      |                             |
| `viewTotal` | Long          | 查看总数      |                             |
| `userName` | String        | 用户名称      |                             |
| `account` | String        | 账号     |                                                                                                                                          |
| `nickname` | String        | 呢称     |                                                                                                                                          |
| `userRoleNameList` | List&ltString>        | 用户角色名称集合     |                                                                                                                                          |
| `avatarPath` | String        | 头像路径     | 页面上显示200 &times; 200头像示例{{question.avatarPath}}{{question.avatarName}}  <br> 显示100 &times; 100头像示例{{question.avatarPath}}100x100/{{question.avatarName}} |
| `avatarName` | String        | 头像名称     |                                                                                                                                          |
| `isStaff` | Boolean       | 是否为员工      |                             |
