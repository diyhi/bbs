-  **接口说明：** 删除点赞
-  **接口地址：** /user/control/deleteLike
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填          | 说明    | 备注                                                                                                                             |
|:-----------|:-----------|:------------|:------|:-------------------------------------------------------------------------------------------------------------------------------|
| `likeId`      | Long       |  按需填写  | 点赞Id  | 只用本参数或下面两个参数  |
| `module`      | Integer       |  按需填写  | 模块  | 10:话题  20:评论  30:评论回复  40:问题  50:答案  60:答案回复                                                                                     |
| `itemId`      | Long       |  按需填写  | 项Id  | 话题Id、评论Id、评论回复Id、问题Id、答案Id、答案回复Id                                                                         |


### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |  |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |  |


### 错误结果 `error`
| 字段 | 类型              | 描述    |
| :--- |:----------------|:------|
| `like` | String          | 点赞错误 |
