-  **接口说明：** 话题解锁用户列表
-  **接口地址：** /user/control/topicUnhideList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;TopicUnhide>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |
| `topicId`      | Long | 是 | 话题Id  |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;TopicUnhide>](#anchor:话题取消隐藏-TopicUnhide)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 话题取消隐藏 `TopicUnhide`
| 字段 | 类型            | 描述      | 备注                               |
| :--- |:--------------|:--------|:---------------------------------|
| `id` | String          | 话题取消隐藏Id    |                                  |
| `userName` | String        | 取消隐藏的用户名称      |                                  |
| `cancelTime` | LocalDateTime | 取消隐藏时间    |                                  |
| `hideTagType` | Integer          | 隐藏标签类型    | 10:输入密码可见  40:积分购买可见   50:余额购买可见 |
| `point` | Long        | 消费积分    | 当字段`hideTagType`等于`40`时有值        |
| `amount` | BigDecimal        | 消费余额    | 当字段`hideTagType`等于`50`时有值        |
| `topicId` | Long | 话题Id  |                                  |
