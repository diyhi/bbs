-  **接口说明：** AI助手
-  **接口地址：** /user/control/aiAssistant
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [Map&lt;String,Object>](#anchor:返回结果)



### 请求参数表
| 字段         | 类型         | 是否必填          | 说明 |
|:-----------|:-----------|:------------|:---|
| `prompt`      | String       |  是  | 提示词  |
| `interfaceProduct`    | Integer     |是 |  接口产品  |
| `captchaKey` | String     | 按需填写 | 验证码编号 |
| `captchaValue`  | String     | 按需填写 |  验证码值 同一编号验证码每分钟提交次数不能超过60次,超出时验证码图片返回空字符  |




### 返回结果
| 字段 | 类型                                          | 描述          | 备注                                                           |
| :--- |:--------------------------------------------|:------------|:-------------------------------------------------------------|
| `success` | boolean                                     | 成功返回`true` 失败返回`false` |                                                              |
| `error` | [Map&lt;String,String>](#anchor:错误结果-error) | 错误值集合 |                                                              |
| `captchaKey` | String                                      |新验证码编号 | 当`success`参数返回`false`时根据'全站设置 - 基本设置 - AI助手每分钟提交超过'参数计算是否含有值 |
| `chatInfo` | [ChatInfo](#anchor:对话信息-ChatInfo)           | 对话信息 | 当`success`参数返回`true`时有值                                      |


### 对话信息 `ChatInfo`
| 字段 | 类型                                          | 描述          | 备注                                    |
| :--- |:--------------------------------------------|:------------|:--------------------------------------|
| `text` | String                                     | 模型生成的回复内容(Markdown格式) |  |
| `interfaceProduct` | Integer| 接口产品 | 10:阿里云百炼大模型  |
| `statusCode` | Integer                                      |表示状态码，调用成功时为空值 ||
| `errorCode` | String                                  | 表示错误码，调用成功时为空值 |  |
| `errorMessage` | String                                  | 表示失败详细信息，成功忽略 |  |



### 错误结果 `error`
| 字段 | 类型              | 描述     |
| :--- |:----------------|:-------|
| `captchaValue` | String          | 验证码错误  | 
| `aiAssistant` | String          | AI助手错误   | 
| `prompt` | String          | 提示词错误 |
| `interfaceProduct` | String          | 接口产品错误 |
