-  **接口说明：** 私信列表
-  **接口地址：** /user/control/privateMessageList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;PrivateMessage>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |



### 分页 `PageView`
| 字段 | 类型                                     | 描述          | 备注                                   |
| :--- |:---------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;PrivateMessage>](#anchor:私信-PrivateMessage) | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)    | 分页索引 |                                      |
| `totalpage` | long                                   | 总页数 |                                      |
| `maxresult` | long                                   | 每页显示记录数 |                                      |
| `currentpage` | long                                   | 当前页 |                                      |
| `pagecount` | long                                   | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 私信 `PrivateMessage`
| 字段 | 类型            | 描述      | 备注                                                                                                                                                                                                       |
| :--- |:--------------|:--------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `id` | String          | Id    |                                                                                                                                                                                                          |
| `friendUserId` | Long          | 私信对方用户Id    |                                                                                                                                                                                                          |
| `friendUserName` | String        | 私信对方用户名称    |                                                                                                                                                                                                          |
| `friendAccount` | String        | 私信对方账号    |                                                                                                                                                                                                          |
| `friendNickname` | String | 私信对方用户呢称    |                                                                                                                                                                                                          |
| `friendAvatarPath` | String | 私信对方头像路径  | 页面上显示200 &times; 200头像示例{{privateMessage.friendAvatarPath}}{{privateMessage.friendAvatarName}}   <br>显示100 &times; 100头像示例{{privateMessage.friendAvatarPath}}100x100/{{privateMessage.friendAvatarName}} |
| `friendAvatarName` | String        | 私信对方头像名称      |                                                                                                                                                                                                          |
| `senderUserId` | Long       | 发送者用户Id      |                                                                                                                                                                                                          |
| `senderUserName` | String        | 发送者用户名称      |                                                                                                                                                                                                          |
| `senderAccount` | String        |  发送者账号     |                                                                                                                                                                                                          |
| `senderNickname` | String        |   发送者用户呢称    |                                                                                                                                                                                                          |
| `senderAvatarPath` | String        |   发送者头像路径    | 页面上显示200 &times; 200头像示例{{privateMessage.senderAvatarPath}}{{privateMessage.senderAvatarName}}   <br>显示100 &times; 100头像示例{{privateMessage.senderAvatarPath}100x100/{{privateMessage.senderAvatarName}}  |
| `senderAvatarName` | String        |    发送者头像名称   |                                                                                                                                                                                                          |
| `messageContent` | String        |    消息内容   |                                                                                                                                                                                                          |
| `status` | Integer        |    消息状态   |          10:未读  20:已读                                                                                                                                                                                                |
| `sendTime` |     LocalDateTime    |   发送时间    |                                                                                                                                                                                                          |
| `readTime` |     LocalDateTime    |   阅读时间    |                                                                                                                                                                                                          |
