-  **接口说明：** 用户登录日志
-  **接口地址：** /user/control/userLoginLogList
-  **请求方式：** GET
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [PageView&lt;UserLoginLog>](#anchor:分页-PageView)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `page`      | Integer | 否 | 页码  |


### 分页 `PageView`
| 字段 | 类型                                                   | 描述          | 备注                                   |
| :--- |:-----------------------------------------------------|:------------|:-------------------------------------|
| `records` | [List&lt;UserLoginLog>](#anchor:用户登录日志-UserLoginLog)                             | 数据展示列表 |   |
| `pageindex` | [PageIndex](#anchor:分页索引-PageIndex)                  | 分页索引 |                                      |
| `totalpage` | long                                                 | 总页数 |                                      |
| `maxresult` | long                                                 | 每页显示记录数 |                                      |
| `currentpage` | long                                                 | 当前页 |                                      |
| `pagecount` | long                                                 | 页码显示总数 |                                      |



### 分页索引 `PageIndex`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `startindex` | long                       | 分页开始索引 |                    |
| `endindex` | long | 分页结束索引 |                        |



### 用户登录日志 `UserLoginLog`
| 字段 | 类型            | 描述         | 备注                                                           |
| :--- |:--------------|:-----------|:-------------------------------------------------------------|
| `id` | String          | Id         |                    |
| `userId` | String        | 用户Id    |                        |
| `logonTime` |    LocalDateTime       | 登录时间         |                        |
| `typeNumber` | Integer          | 类型编号         |          10:登录 20:续期              |
| `ipAddress` | String          | IP归属地       |                        |
