-  **接口说明：** 会员卡明细
-  **接口地址：** 自定义
-  **请求方式：** 自定义
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [MembershipCard](#anchor:会员卡-MembershipCard)


### 请求参数表
| 字段         | 类型     | 是否必填 | 说明 |
|:-----------|:-------|:-----|:---|
| `membershipCardId`      | Long | 是    | 会员卡Id  |




### 会员卡 `MembershipCard`
| 字段 | 类型              | 描述      | 备注                                                           |
| :--- |:----------------|:--------|:-------------------------------------------------------------|
| `id` | Long            | 会员卡Id    |                    |
| `name` | String          | 名称      |                        |
| `subtitle` | String          | 副标题    |                       |
| `userRoleId` | String          | 角色Id    |                       |
| `lowestPrice` | BigDecimal      | 最低销售价      |                        |
| `highestPrice` | BigDecimal      | 最高销售价      |                        |
| `lowestPoint` | Long            | 最低积分      |                        |
| `highestPoint` | Long            | 最高积分      |                        |
| `createDate` | LocalDateTime   | 创建时间    |                        |
| `state` | Integer         | 状态      |    1:上架  2.下架                    |
| `descriptionTagList` | List&lt;String> | 说明标签集合      |                        |
| `specificationList` | [List&lt;Specification>](#anchor:规格-Specification)    | 规格集合      |                        |
| `sort` | Integer         | 排序      |            默认为0            |
| `introduction` | String          | 简介    |                       |



### 规格 `Specification`
| 字段 | 类型                            | 描述          | 备注                                                           |
| :--- |:------------------------------|:------------|:-------------------------------------------------------------|
| `id` | long                       | 规格Id |                    |
| `specificationName` | String | 规格名称 |                        |
| `membershipCardId` | long |  会员卡Id|                        |
| `stock` | long | 可用库存量 |                        |
| `point` | long | 支付积分 |                        |
| `marketPrice` | BigDecimal | 市场价 |                        |
| `sellingPrice` | BigDecimal | 销售价 |                        |
| `duration` | Integer | 时长 |                        |
| `unit` | Integer | 时长单位 |       10.小时 20.日 30.月 40.年                 |
| `sort` | Integer | 排序 |                        |
