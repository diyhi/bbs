-  **接口说明：** 查询微信openid
-  **接口地址：** /thirdParty/queryWeiXinOpenId
-  **请求方式：** POST
-  **请求参数：** [参数表](#anchor:请求参数表)
-  **返回类型：** [WeiXinOpenId](#anchor:返回结果-`微信openid-WeiXinOpenId`)


### 请求参数表
| 字段         | 类型         | 是否必填 | 说明                                                 |备注                                        |
|:-----------|:-----------|:-----|:---------------------------------------------------|:---------------------------------------------------|
| `code`      | String       | 是    | 换取access_token的票据    | code作为换取access_token的票据，每次用户授权带上的code将不一样，code只能使用一次，5分钟未被使用自动过期 | 



### 微信openid `WeiXinOpenId`
| 字段 | 类型                                          | 描述          | 备注                                        |
| :--- |:--------------------------------------------|:------------|:------------------------------------------|
| `openId` | String                                      | 用户的唯一标识 |              每个用户对每个公众号的OpenID是唯一的。对于不同公众号，同一用户的openid不同                             |
| `errorCode` | String                                            | 错误代码 |                                           |
| `errorMessage` | String | 错误信息 |                                           |
